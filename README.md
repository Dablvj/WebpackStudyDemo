## 项目模板
- [React Starter Kit](http://www.reactstarterkit.com/): 基于 React, Node.js, Express 和 Flux 架构的 Web 开发模板
- [react-redux-universal-hot-example](https://github.com/erikras/react-redux-universal-hot-example): 基于 express, react, redux, webpack, react-transform 的统一 web 应用模板
- [react-transform-boilerplate](https://github.com/gaearon/react-transform-boilerplate): 基于 webpack, 支持热加载 react 组件的项目模板
- [react-starter](https://github.com/webpack/react-starter): 基于 webpack 开发 react 项目模板
- [angular2-webpack-starter](https://github.com/AngularClass/angular2-webpack-starter): Angular2 和 Webpack 项目模板
- [generator-react-webpack](https://github.com/newtriks/generator-react-webpack): 基于 React 和 Webpack 的项目 Yeoman 生成器
- [hjs-webpack](https://github.com/HenrikJoreteg/hjs-webpack): 支持热加载 React 组件的 webpack 配置工具
- [react-webpack-rails-tutorial](https://github.com/shakacode/react-webpack-rails-tutorial): 集成 rail, react, redux, webpack 的示例
- [css-modules/webpack-demo](https://github.com/css-modules/webpack-demo): CSS 模块化示例
- [fullstack-react](https://github.com/Widen/fullstack-react): 基于 react, webpack, falcor 的全栈 javascript 单页应用
- [react-webpack-node](https://github.com/choonkending/react-webpack-node): 基于 webpack 的同构 React 和 Flux 项目模板
- [react-static-boilerplate](https://github.com/koistya/react-static-boilerplate): React 应用模板和工具
- [gaearon/library-boilerplate](https://github.com/gaearon/library-boilerplate): 包含 ESLint, Mocha, Babel, Webpack, 并支持热加载的 React 项目模板
- [koa-react-full-example](https://github.com/dozoisch/koa-react-full-example): 使用 Koa, React, Passport, Mongoose, Webpack, Mocha, Babel 开发项目的一个完整示例
- [wbkd/react-starterkit](https://github.com/wbkd/react-starterkit):  react-router, reflux, jest, webpack, gulp and stylus 的 React 项目模板
- [webpack-express-boilerplate](https://github.com/christianalfoni/webpack-express-boilerplate): 一个在 node express 中运行 webpack 的项目模板
- [react-webpack-server-side-example](https://github.com/webpack/react-webpack-server-side-example): 使用 webpack, 且支持服务端渲染的 React 示例
- [vue-webpack-example](https://github.com/vuejs/vue-webpack-example): Vue.js + Webpack 示例
- [shanewilson/react-webpack-example](https://github.com/shanewilson/react-webpack-example): 使用 webpack 构件, Jest 测试 的 React 示例

- [webpack-isomorphic-tools](https://github.com/halt-hammerzeit/webpack-isomorphic-tools): 服务端渲染 webpack 构件程序
